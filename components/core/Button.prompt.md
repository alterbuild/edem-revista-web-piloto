Primary EDEM button — use for the main call-to-action in any view; squared editorial corners.

```jsx
<Button variant="primary" size="lg">Leer Revista Digital</Button>
<Button variant="outline">Ver ediciones</Button>
```

Variants: `primary` (teal), `secondary` (ink), `outline`, `ghost`. Sizes: `sm` `md` `lg`. Pass `iconLeft` / `iconRight` for Lucide icons, `as="a"` for links.
